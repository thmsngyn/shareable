## Spotify share feed project

React app with a spotify integration.

## Requirements

- NPM
- Heroku

## Dev setup

```
npm ci
```

```
npm run start:dev
```

## Deploy

### Deploy to Heroku app

Simply push changes to master and changes will be automatically deployed

### Deploy to Git hub pages (deprecated)

```
npm run deploy:prod
```
